全米トレンド波及分析
1. 分析の背景・目的

背景: 2026年4月20日、NBAの「76ers（フィラデルフィア・セブンティシクサーズ）」に関する検索ワードが全米で急上昇した。
目的: 特定のスポーツイベントが、チームの本拠地（ペンシルベニア州）以外にどのように波及しているかを可視化し、デジタル広告の配信優先エリアを特定する。
2. 分析結果の解釈（インサイト）

意外な発見: 上昇率上位に、本拠地から遠い Amarillo（テキサス州） や Las Vegas（ネバダ州）、さらには Portland（オレゴン州） などの地域がランクインしている。
考察: 平均上昇率（avg_percent_gain）が 427.5% と非常に高く、全米各地で一斉に注目を浴びている。これは、対戦相手の人気や、プレーオフなどの重要局面が全国的な関心を集めていることを示唆している。
ビジネス提案: 本拠地周辺だけでなく、これらの急上昇地域に対してもチームグッズのオンライン販売や、試合のライブ配信サービスの広告を強化することで、効率的なコンバージョン（成約）が期待できる。
3. 使用したスキル

BigQuery: 大規模なパブリックデータ（Google Trends）の操作。
SQL: LIKE による柔軟なキーワードフィルタリング、GROUP BY を用いた地域別集計、AVG や ROUND による指標の定量化。
