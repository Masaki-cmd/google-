全米トレンド波及分析
1. 分析の背景・目的

背景: 2026年4月20日、NBAの「76ers（フィラデルフィア・セブンティシクサーズ）」に関する検索ワードが全米で急上昇した。
目的: 特定のスポーツイベントが、チームの本拠地（ペンシルベニア州）以外にどのように波及しているかを可視化し、デジタル広告の配信優先エリアを特定する。
2. 分析結果の解釈（インサイト）

意外な発見: 上昇率上位に、本拠地から遠い Amarillo（テキサス州） や Las Vegas（ネバダ州）、さらには Portland（オレゴン州） などの地域がランクインしている。
考察: 平均上昇率（avg_percent_gain）が 427.5% と非常に高く、全米各地で一斉に注目を浴びている。これは、対戦相手の人気や、プレーオフなどの重要局面が全国的な関心を集めていることを示唆している。
ビジネス提案: 本拠地周辺だけでなく、これらの急上昇地域に対してもチームグッズのオンライン販売や、試合のライブ配信サービスの広告を強化することで、効率的なコンバージョン（成約）が期待できる。
3. 使用したスキル

BigQuery: 大規模なパブリックデータ（Google Trends）の操作。
SQL: LIKE による柔軟なキーワードフィルタリング、GROUP BY を用いた地域別集計、AVG や ROUND による指標の定量化。

NBAプレーオフ進出チームの全米関心度比較分析
1. プロジェクト概要
目的: NBAの人気チーム（76ers vs Celtics）の検索トレンドを比較し、地域ごとのファン層の厚みと、トレンドの波及範囲を特定する。
使用データ: bigquery-public-data.google_trends.top_rising_terms (Googleトレンド 公開データ)
2. 分析手法
SQLによる多角分析:
CASE 文と AVG 関数を組み合わせた、同一クエリ内での競合指標比較。
LIKE 演算子を用いた柔軟なキーワードマッチング。
地域（DMA）単位での集計によるマイクロマーケティング視点の導入。
3. 主要なインサイト（結果の解釈）
Celticsの圧倒的優位: 分析したほぼすべての地域において、Celticsの上昇率（平均 459.2%）が 76ers（平均 427.5%）を上回った。
広域トレンドの発生: 本拠地（ボストン、フィラデルフィア）以外のテキサス、アラスカ、ジョージアなど、全米各地で高い上昇率を記録。これは、特定地域に限定されない「全国的な注目イベント」であることを示している。
ビジネスアクション案:
上昇率の高い Celtics 優勢エリアに対して、Celtics 関連グッズの重点的な在庫配分を提案。
76ers 優勢な地域がもし見つかれば、そこを「逆転の余地があるターゲット市場」として、集中的なファン獲得キャンペーンを実施する。
4. 技術スタック
Google BigQuery: ペタバイト規模のデータを瞬時に集計。
GoogleSQL: 複雑な条件分岐を含む集計クエリの設計。
